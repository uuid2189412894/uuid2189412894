# pilleater Website

A 2000s cyberpunk-style website built with HTML4 elements and modern Tailwind CSS.

## Features

- HTML4 compliant structure using tables and traditional elements
- Modern responsive design with Tailwind CSS
- Cyberpunk aesthetic with glitch effects and neon colors
- Mobile-friendly navigation
- All content from rentry.co/pilleater

## Setup

1. Place `index.html` and `styles.css` in the same directory
2. Open `index.html` in a web browser
3. No build process required - uses CDN for Tailwind CSS

## Browser Support

Works in all modern browsers. Best viewed with Chrome, Firefox, or Edge.

## Notes

- Uses HTML4 DOCTYPE and table-based layouts
- Responsive design adapts to mobile devices
- Custom CSS adds cyberpunk effects (glitch, scanlines, etc.)
- Footer includes "WEBSITE OPERATED BY 3DTESTOSTERONE"
